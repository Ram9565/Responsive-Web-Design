Introduction

JavaScript is a versatile language used for web development.

It runs in the browser and allows for dynamic content.

It was originally created in 10 days!

console.log('Hello, world!');

JavaScript can also be used on the server side via Node.js.

  • Dynamic content
  • Event handling
  • Browser API access
Variables

You can declare variables using var, let, or const.

`let` is block scoped and preferred for mutable values.

`const` is used for values that donâ€™t change.

let x = 10; const pi = 3.14;
  • let is block scoped
  • const cannot be reassigned
  • var is function scoped

Always try to use const when possible.

Functions

Functions are blocks of code designed to perform tasks.

They can accept inputs (parameters) and return outputs.

function greet(name) { return "Hello " + name; }

Functions can be anonymous or named.

Arrow functions are a modern syntax.

const add = (a, b) => a + b;
  • Reusable code
  • Encapsulation
  • Modularity
Loops

Loops are used to repeat code.

JavaScript supports for, while, and do-while loops.

for(let i = 0; i < 5; i++) { console.log(i); }

Always ensure loops terminate to avoid infinite loops.

Use `break` to exit a loop early.

  • for loop
  • while loop
  • do-while loop
Arrays

Arrays are ordered collections of values.

You can access elements using indexes starting at 0.

JavaScript provides many array methods.

let fruits = ['apple', 'banana']; fruits.push('orange');
  • push()
  • pop()
  • map()

Arrays can contain any data type including other arrays.
